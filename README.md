# mk’s cams (by graispect)

an ios film camera app with custom lenses, glitch filters, and  
a minimal interface inspired by old disposable cameras.

### features

- live filters.  
- custom core image pipeline.
- deterministic seeded randomness.  
- save filtered versions only.  
- built-in gallery. 
- auto-create a photos album.    (“mk's cams”)
- all icons & ui designed by yours truly.

### what i did

- designed the ui & icons through procreate. 
- built the camera session.  
- created every filter by being inspired of old cameras and disposables.
- created every filter by core image overlayed.
- designed the gallery experience.
- implemented storage & permissions flow.
